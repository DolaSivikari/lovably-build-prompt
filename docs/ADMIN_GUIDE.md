# Admin Panel User Guide

## Getting Started

### Login
Access the admin panel at `/admin` and sign in with your admin credentials.

### Dashboard Overview
The admin dashboard provides quick access to:
- Content management (Services, Projects, Blog Posts)
- Inbox (Contact forms, RFPs, Resumes, Prequalifications)
- Settings (Homepage, Footer, Contact, About, Site)
- Media Library
- User Management

## Content Management

### Hero Management
**Hero Slides**: Manage rotating hero carousel on homepage
**Hero Images**: Upload hero images for specific pages

### Services
- Create/edit service offerings
- Set publish status (draft, published, archived)
- Add SEO metadata
- Bulk actions: delete, status change

### Projects
- Showcase portfolio projects
- Upload before/after images
- Associate with services
- Feature projects on homepage

### Blog Posts
- Write articles and case studies
- Rich text editor with formatting
- Category and tag management
- Preview before publishing

## Settings Pages

### Homepage Settings
- Hero carousel configuration
- Featured sections
- Call-to-action buttons

### Footer Settings
- Company information
- Social media links
- Legal links

### Contact Page Settings
- Office addresses
- Phone numbers
- Email addresses
- Business hours

### Site Settings
- Global site configuration
- Meta information
- Analytics setup

## Tips & Shortcuts
- Use tooltips (?) for field help
- Preview changes before publishing
- Bulk select with checkboxes
- Sort tables by clicking headers
- Filter and search for quick access
